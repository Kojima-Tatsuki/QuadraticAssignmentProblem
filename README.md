# 二次割り当て問題(QAP)

## 例
以下のような配置の場合の例を示す.  
順列 $q = \\{ 2, 1, 4, 3 \\}$ のように表現される.

| 施設 | 場所 |
| :--: | :--: |
| 2 | 1 |
| 1 | 2 |
| 4 | 3 |
| 3 | 4 |
